
# Aesthetic Recipe Finder

A responsive API-powered recipe search app built with HTML, CSS, and JavaScript using TheMealDB API.

## Features
- Search recipes by name
- Dynamic recipe cards
- Recipe detail modal/view
- Loading state
- Error handling
- Responsive design

## API
TheMealDB API: https://www.themealdb.com/api.php
